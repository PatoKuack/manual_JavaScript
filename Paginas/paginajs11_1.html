Scope.

Scope o alcance.

Es el alcance que puede tener una variable en tu código.

Scope Local o Variable Locar se refiere a la variable o función que está dentro de un bloque o función especifica. Solo se pueden acceder a ellas (ejecutar o llamar) dentro del entrono en donde conviven.

const fruits = () => {
var fruit = 'apple';
console.log(fruit);
//se declara e imprime el valor de la variable fruit dentro de la función.
}
fruits();
//imprime: apple
console.log(fruit);
//imprime un error de no asignación ya que la variable "fruit" solo existe dentro del bloque de código de la función donde se declaró.

Las variables tienen un scope local diferente dependiendo del ámbito en el que se encuentran como por ejemplo, var tiene un ámbito de función, es decir que las variables declaradas con var tienen un alcance (existen) en toda la función, pero las variables declaradas con let y const tienen un ámbito de bloque por lo que tienen un alcance (existen) sólo en el bloque en el que se les declara, ejemplo:

const fruits = () => {
if (true) {
var fruit1 = 'apple';
let fruit2 = 'banana';
const fruit3 = 'melon';
console.log(fruit2);
console.log(fruit3);
//se declaran las variables.
//se imprime la variable declarada con let y const porque estas variables solo existirán en este fragmento de bloque.
}
console.log(fruit1);
//se puede imprimir la variable declarada con var fuera del bloque donde se declaró porque existe dentro de toda la función.
}
fruits();
//imprime: banana después melon y por último apple

Scope Global o Variable Global se refiere a la variable o función que está afuera de cualquier bloque o funcion. Se pueden acceder a ellas (ejecutar o llamar) en cualquier parte del entrono dentro del código.

La reasignación de valores en las variables se hace en cascada y es recomendable utilizar constantes en vez de variables (var o let) globales. También es recomendable no utilizar variables con el mismo nombre dentro y fuera de bloques de código ya que se pueden modificar los valores de las variables como en el siguiente caso:

let nombre = "Deiv"
const saludar = function() {
nombre = "Alex"
console.log(`${nombre}`)
}
saludar()
//imprime: Alex (porque así lo marca esta función)
 console.log(nombre)
//imprime: Alex (porque se llamó a la variable global "nombre" dentro del bloque de código y se le asignó un nuevo valor)

Las constantes globales pueden ser modificadas en bloques de código declarando una constante con el mismo nombre, pero sólo funcionarán dentro de ese bloque de código, ejemplo:

const nombre2 = "Deiv"
const saludar = function() {
const nombre2 = "Alex";
console.log(`${nombre2}`)
}
saludar();
//imprime Alex (porque así lo marca esta función)
 console.log(nombre2);
//imprime Deiv (porque el valor "Alex" de la variable "nombre2" se creó y existió solamente dentro del bloque de código de la función y aunque se quisiera asignar un nuevo valor a "nombre2" no se podría porque es una constante).

Lexical Scope o Ámbito Léxico.

El ámbito léxico se refiere a que una función puede acceder a una función o variable fuera de ella. Cada nivel interno puede acceder a sus niveles externos hasta poder alcanzarlas.

var scope = "Soy Localin";

return scope;
//dentro de una función se retorna la variable con su valor.
