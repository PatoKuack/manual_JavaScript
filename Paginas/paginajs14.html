CSSOM

Es el modelo de objeto de CSS, es equivalente al DOM pero en CSS. Esto permite modificar CSS desde JavaScript.

style

Es una propiedad que permite acceder a los estilos de un elemento HTML, de un elemento en el DOM. Con la nomenclatura del punto y la propiedad CSS que se quiera modificar, se puede trabajar con los estilos de CSS en JavaScript. Con el detalle que, si la propiedad en CSS lleva guiones, deberás utilizar camelCase.

const title2 = document.querySelector('h2');
title2.style.backgroundColor = 'red';
// En CSS el estilo se escribe: background-color: red;

matchMedia()

Permite evaluar breakPoints y utilizar mediaQuerys de CSS las cuales permiten determinar si la página está en horizontal o vertical, adaptan su ancho o largo entre otros veneficios.

const breackPoint = matchMedia('(min-width: 640px) and (orientation: portrait)');
// matchMedia evalua una mediaQuery de CSS y devolverá un valor booleano si se cumple con el tamaño y orientación en este caso.
const colorear = () => {
 if(breackPoint.matches){
  document.body.style.background = 'pink';
 } else {
  document.body.style.background = 'purple';
} }
// Se encapsulan las aplicaciones del estilo para utilizarlas en eventos, al cargar o modificar el tamaño de la página.
addEventListener('DOMContentLoaded', colorear);
// Aplica el evento al cargar la página.
addEventListener('resize', colorear);
// Cuando se modifique el tamaño del navegador se aplicará el evento.

getComputedStyle()

Esta función permite obtener los estilos calculados de un elemento.

Un estilo calculado es el resultado de su modificación en CSS ya en el navegador, como por ejemplo el tamaño de fuente de un artículo al que se le dio un font-size de 2em (el doble de su tamaño original), obtendríamos ese tamaño duplicado en vez del original.

console.log(getComputedStyle(document.body).font-zise);
// Nos regresaría un valor como "64px", que sería el tamaño de la fuente.
console.log(getComputedStyle(document.body).position);
// Nos regresaría un valor como "static", que sería la posición del elemento.

getBoundingClientRect()

Este método devuelve las dimensiones y posiciones exactas de un elemento como el "bottom: ..." que indica cuanto espacio hay del borde superior de la página a la esquina superior izquierda (en perspectiva con el usuario) del elemento (top, left y right también toman su referencia con los bordes de la página y la misma esquina mencionada del elemento) o el "width: ..." o las coordenadas en "X" o en "Y".

// Desde la consola del navegador:
$0.getBoundingClientRect();
// Nos regresaría un DOMRect{...} el cual contiene los detalles de la posición de un elemento.
$0.getBoundingClientRect().height;
// Nos regresaría el valor directo que corresponde a la altura del elemento especificado.

naturalWidth y naturalHeight

Estas propiedades devolverán el tamaño real y autentico de una imagen, no el que ha sido modificado por código para dar un estilo.
