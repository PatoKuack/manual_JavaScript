Operadores

Operadores de aritméticas

Los operadores aritméticos (+, -, *, /, %, **) permiten crear operaciones matemáticas en la declaración de un valor, ejemplo:
let num = 5-15

Operadores de asignación

El operador = sirve para asignar o reasignar o asignar por referencia un valor a una variable.
Se puede reducir asignaciones combinando operadores aritméticos y de asignación, ejemplo:
a %= b, en este caso se reasigna el residuo de a entre b a la variable a.
Lo mismo se puede hacer con cualquier operador aritmético colocándolo en lugar de '%'.

Operadores de concatenación

El operador + concatena strings, ejemplo:
let nomCompleto = nombre + " " + apellido

Operador de comparación.

Comparan valores y devuelve true o false según se cumpla o no la comparación.
Igual que == compara solo valores (un String "5" puede ser igual que un Number 5).
Estrictamente igual que === compara valor y tipo de dato.
Diferente que != compara solo valores (un String "5" no es diferente que un Number 5).
Estrictamente diferente que !== compara valor y tipo de dato.
Menor que < se usan estrictamente para valores numéricos.
Menor o menor que <= se usan estrictamente para valores numéricos.
Mayor que > se usan estrictamente para valores numéricos.
Mayor que >= se usan estrictamente para valores numéricos.
in devuelve un valor booleano si la propiedad especificada está en el objeto especificado, ejemplo:
animal = ["oso", "perro", "albatros"] //true con arreglos
0 in animal //devuelve true porque hay una posición 0 en el array
animal = {"oso", "perro", "albatros"} //true con objetos
"oso" in animal //devuelve true porque hay un string en el objeto

Operadores lógicos

And && concatena comparaciones pidiendo que se cumplan todas las comparaciones involucradas
Or || concatena comparaciones pidiendo que se cumplan al menos una de las comparaciones involucradas
Es buena práctica separar las comparaciones con paréntesis, ejemplo:
(b > a) || (b > c)

Operadores unarios

Necesitan de ellos mismos para modificar su valor.
Incremento en uno ++, a=a+1 == a++
Decremento en uno --, a=a-1 == a--

Operadores ternarios

Asigna un valor1 a una variable si se cumple una condición, sino le asigna un valor2, ejemplo:
let c = 30 > 21 ? valor1 : valor2

Type coercion

Capacidad que tiene JavaScript para convertir un tipo de dato en otro al realizar interacciones entre variables con tipos de datos diferentes, ejemplo:
let numA = "1"; //numA será de tipo "string"
let suma = 3 + numA;
//suma="31"; convirtió el número 3 en string y lo concatenó con la variable numA.
let numB = "5"; //numB será de tipo "string"
let resta = numB - 2;
//resta=3; convirtió el string "5" en número y se le restó la variable numB.
let resta = 4 - "perro";
//resta=NaN; el navegador espera de resultado un numero por tratarse de una resta así que el resultado será de tipo número, sin embargo, como no se puede realizar la resta el resultado será un Not-a-Number.
let suma = true + true;
//sume =2; como en lógica binaria true = 1 y false = 0, entonces el navegador convierte estos dos valores de tipo booleanos a tipo numérico y los suma.

Converción de tipo de valores.

parseInt("2.99")
Convierte cualquier número en un entero, ya sea que sea un numero con fracción o un numero de tipo string. En un numero con fracción trunca sus decimales, en el ejemplo de 2.99 se convertiría en 2.
Object.entries(array)
Convierte una lista de valores (array) en una lista de objetos.
Array.from(object)
Convierte una lista de objetos en una lista de valores (array).
