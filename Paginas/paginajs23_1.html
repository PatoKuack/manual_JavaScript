DOM traversing.

DOM traversing significa recorrer el DOM es decir, obtener un elemento HTML y poder recorrerlo por los hermanos, padres, e hijos de este elemento.

Seleccionar elementos.

Hijos:

childNodes
Devuelve todos los nodos hijos (etiquetas, textos, saltos de línea entre etiquetas, etc.) de un nodo padre.
children
Devuelve todos los elementos hijos (etiquetas) de un nodo padre.
firstChild
Devuelve el primer nodo hijo de un nodo padre (no necesariamente devuelve un elemento, podría ser texto o elemento).
firstElementChild
Devuelve el primer elementos hijo de un nodo padre.
lastChild
Devuelve el último nodos hijo de un nodo padre.
lastElementChild
Devuelve el último elementos hijo de un nodo padre.
hasChildNodes()
Devuelve un booleano dependiendo de que el nodo padre tenga hijos nodos o no (un espacio o salto de línea entre la apertura y cierre de la etiqueta cuenta como un nodo).

Hermanos:

nextSibling
Devuelve el siguiente nodo hermano de un nodo.
nextElementSibling
Devuelve el siguiente elemento hermano de un nodo.
previousSibling
Devuelve el anterior nodo hermano de un nodo.
previousElementSibling
Devuelve el anterior elemento hermano de un nodo.

Padres:

parentNode
Devuelve el nodo padre de un elemento hijo.
parentElement
Devuelve el elemento padre de un elemento hijo.
.parentNode.parentNode || .parentElement.parentElement
Se pueden anidar para obtener el nodo o elemento padre de un nodo nieto.
