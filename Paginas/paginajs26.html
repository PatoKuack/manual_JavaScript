Persistencia de datos.

localStorage.

local.storage es una herramienta de los navegadores que ayuda a guardar información (en forma de string) de la página, esta información no expira aunque se apague el dispositivo que se utiliza y esta disponible en la web de forma global. Esta herramienta también permite utilizar su información guardada para interactuar con el usuario.

Existe otra herramienta para almacenar información, session.storage solo esta disponible en la ventana del navegador actual y si se cierra la ventana se pierde su información.

Ver contenido del LS.

Desde la consola del navegador se puede escribir localStorage para visualizar la información que se tiene guardada.

Transformar a texto.

Debido a que localStorage solo almacena información en texto, para poder guardar la información que se desee, se utiliza la función JSON.stringify() . Se le coloca como argumento los datos que se quieran transformar a string como objetos, arrays o array de objetos, ejemplo: JSON.stringifly([{text: 'mytext', completed: 'false'}])

Transformar a código.

Para transformar el contenido del localStore a un formato de código como en objetos o en arrays, se utiliza la función JSON.parse() en la cual se le coloca como argumento el string que se desea transformar.

Insertar datos en LS.

Para insertar información en el localStorage, se utiliza la instrucción localStorage.setItem('variable', 'valor'). Como primer parámetro se coloca la variable en forma de string que almacenará el valor y como segundo parámetro se coloca el valor, también en forma de string.

Si se guarda una variable que ya existe, los datos almacenados en la variable existente serán reemplazados por los datos de la variable que se guardará.

Obtener variable de LS.

Para obtener una variable del localStorage se utiliza la instrucción localStorage.getItem('variable').
