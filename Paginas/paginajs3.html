Condicionales

if-else

Realiza una acción si la condición se cumple if ( condición ){ acción }
Si la condición no se cumple, el programa hace caso omiso a la sentencia if.
Puede realizarse una segunda acción en caso de que la condición no se cumpla colocando en seguida el comando else { acción 2 }
Se puede anidar una condición utilizado un else if{}, para anclar un segundo if si la condición del primer if no se cumple.
Para una condición booleana, se puede especificar si la condición es true o false
o se puede escribir solamente la variable, el programa entiende que si es verdadera se cumple la condición y si es falsa no se cumple, ejemplo:
let pepinillo = false
let escondidos = true
if (pepinillo){ acción 1 }
else if (escondidos === true){ acción 2 }
else { acción3 }
Se puede colocar simplemente if (variable) para preguntar si la variable existe, es decir, si se ha introducido alguna variable o if (!variable) para preguntar si la variable no existe, es decir, si el campo donde se coloca la variable está vacío.
Se puede colocar una sentencia if en una línea, ejemplo:
if (condición) return acción
Otra forma de utilizar if y else es de la siguiente forma:
var variable = condición ? "acciónTrue" : "acciónFalse";

switch

Se pueden utilizar condiciones múltiples para validar una variable con la palabra reservada switch, en función de esa variable se determina la acción a ejecutar,
para esto se especifican los casos que se desean tomar en cuenta por si el valor de la variable entra en alguno de ellos con el comando case valor :,
se puede colocar al final un default: para realizar una acción en caso de que el valor de la variable no sea validada por ninguno de los casos previos, ejemplo:
switch( parámetro o condición ){
 case valor1:
  acción
  break;
 default:
  acción
}

Ciclos (bucles o loops).

For

Repite la ejecución de bloques de código contenidos en el comando for hasta que se cumple una condición, entonces se cierra el ciclo y continua el programa,
En el ciclo for se colocan 3 parámetros, el valor de inicialización, la condición y el incremento del valor, ejemplo:
for (let i=0; i>10; i--){
 acción
}
Otra forma de utilizar un for es de la siguiente forma:
for(var variable of array){
 acción
}
La "acción" se aplicará a cada uno de los valores del "array" indicado (el número de ciclos será el número de datos del array) y en cada ciclo la "variable" tomará el valor de los arrays en forma secuencial.

While

Repite la ejecución de bloques de código contenidos en el comando while mientras se cumpla la condición especificada,
se tiene que colocar una acción que convierta la condición de verdadera a falsa para romper el ciclo y continuar con el código, ejemplo:
let vidas = 5
while(vidas > 0){
 vidas--
 acción
}
