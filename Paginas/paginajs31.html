NPM.

NPM o node package manager, es un gestor de paquetes, el más popular que tiene JavaScript, donde se encuentra una gran cantidad de recursos para poder implementar en los proyectos. También permite crear paquetes propios y compartirlos con toda la comunidad.

Instalación de Node en Windows.

Para instalar NPM en Windows es necesario ir a la página de node: nodejs.org, para instalar node y la versión de NPM.

En la página se encuentran dos opciones:

  • La LTS que brinda un soporte a todos los features (características) que están añadidos en esa versión y es mejor para entornos de desarrollo y producción.
  • La Current (que es la que se descargará e instalará) contiene todos lo features que han salido hasta ahora.

Una vez instalado (dando en toda la instalación NEXT), se procede a abrir el entorno del software que es una terminal de comandos. En la terminal se utilizarán las siguientes instrucciones:

node -v
Muestra la versión que se tiene de node.
npm -v
Muestra la versión que se tiene de npm.
npm install -g npm@latest
Comprueba si existe una versión más resiente de npm y si es así se instala en el sistema.
