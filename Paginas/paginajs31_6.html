Seguridad.

Se debe garantizar de que el proyecto no contenga softwares maliciosos y que cumpla con las normas de trabajo. Para evitar ésto se puede utilizar NPM para analizar el proyecto y detectar vulnerabilidades.

npm audit
Sirve para ver las vulnerabilidades del proyecto.
npm audit --json
Genera un archivo .json con información más detallada de lo que está pasando con los paquetes instalados.

Una ves sepamos cual es la vulnerabilidad se procede a actualizar cualquiera de los paquetes.

npm update paquete --depth 2
Instala todas las dependencias del archivo en un nivel de profundidas de 2.
npm audit fix
Soluciona las vulnerabilidades del proyecto y actualiza a la última versión los paquetes con las dependencias que requieren.

Se puede correr de nuevo el comando npm audit para verificar que ya no haya vulnerabilidades. Si aún hay vulnerabilidades, es recomendable actualizar uno por uno, paquete por paquete.

snyk.io es una herramienta que garantiza estar siempre actualizados con las dependencias del proyecto.
