bcrypt

La función de cifrado de bcrypt permite construir una plataforma de seguridad utilizando contraseñas encriptadas y poder verificar la autenticidad de la contraseña.

Primero es necesario instalarla a través de la terminal con la instrucción npm install bcrypt

 const bcrypt = require("bcrypt");
 const password = "NuncaParesDeAprender";
 bcrypt.hash( password, 5, function( err, hash ){
  console.log(hash);
 } );
// El primer parámetro es la contraseña, el segundo es la cantidad de veces que se aplicará el hash y el tercero es la función que retorna.
// Un hash es aplicar un algoritmo para poder cambiar el texto por una cadena de caracteres que no tenga nada que ver con la contraseña inicial.
// el hash entrega una contraseña distinta en cada ejecución del bcrypt, para evaluar si una contraseña concuerda con un hash se utiliza un compare.
 bcrypt.compare( password, hash, function( error, result ){
  console.log(result);
  console.log(error);
 } );
// Devuelve true o false en el result.

Moment.js

Moment es una librería que permite trabajar con fechas y manejar tiempos fácilmente.

Primero es necesario instalarla a través de la terminal con la instrucción npm install moment

 const moment = require('moment')
 const ahora = moment();

// Para ver las opciones que tiene moment:
 console.log(ahora);

// Para ver la fecha:
 console.log(ahora.toString());

// Para darle formato a la fecha:
 console.log(ahora.format('MM/DD/YYYY HH:MM A')); // 04/11/2021 20:10 PM

// Para validad una fecha
 console.log(moment('2021-08-10').isValid()); // Retorna un true o false dependiendo de si la fecha es válida o no

// Para encontrar cuanto tiempo ha pasado hasta hoy
 console.log(moment('2018-04-11').fromNow()); // Hace 2 años

// Para agregar o eliminar años, días o meses
 console.log(moment('2021-08-10').add(1, 'years')); // 2021-04-11
 console.log(moment('2021-08-10').subtract(1, 'years')); // 2019-04-11

Sharp

Sharp trabaja con la librería de manejo de imágenes en c++ que permite convertir imágenes grandes en imágenes JPEG, PNG más pequeñas o compatibles con la web de diferentes dimensiones.

 const sharp = require('sharp')
// Reducira una imagen de cualquier tamaño a 80x80 y la guardara en una imagen más pequeña sin eliminr la original.
 sharp( 'imagen.png' )
// Si a resize se le coloca solo un número se tomará como porcentaje, pero si se colocan 2 los tomará como el largo y el ancho.
  .resize( 80, 80 )
// Convierte la imagen en blanco y negro.
  .grayscale()
// Guarda la imagen modificada en tamaño y color en una imagen nueva con un nuevo nombre especificado.
  .toFile( 'imagen80x80.png' );
