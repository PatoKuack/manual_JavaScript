Funciones.

Las funciones son bloques de código que son reutilizables y pueden ser ejecutados desde cualquier lugar de nuestro código las veces que deseemos.

Las funciones se declaran con la instrucción function seguido del nombre de la función y un bloque de parentesis, después se abre un bloque de llaves para colocar instrucciones que se ejecutarán cuando se mande a llamar la función, ejemplo:
function función(){
código;
}

Las funciones se mandan a llamar simplemente colocando el nombre de la función y un bloque de parentesis, ejemplo:
función();

function saludar(){
console.log("Holiholilla");
}
 //para mandar a llamar la función:
saludar();

Parámetros.

Elemento de la función que es definido al mandar a llamar la función para ser ejecutado, no son esenciales y la variable en donde se almacenan los datos del parámetro sólo existe dentro de la función.
Si se le asigna un valor al parametro, va a significar que cuando el parámetro no tenga algún valor (en caso de que se reciba el valor directamente de algún usuario) adquirirá ese valor por defecto. Sus declaraciones son de la siguiente forma:
function función = valor ( parámetro ) {
bloque de código usando el parámetro
}
Se llaman de la siguiente forma:
función ( Dato(s) del parámetro ) ;
Se puede llamar directamente dentro del código o en la consola. function miFuncion(suma1, suma2){
console.log(suma1 + suma2);
}
miFuncion(247, 585);
 //imprimiría: 832

Argumento.

Representa el valor que se pasa al parámetro de una función cuando se le llama a travez de un código. *Las funciones reciben parámetros y los argumentos se envían a las funciones.

Valores de retorno.

Retorna un valor con la palabra reservada return para utilizar esos valores directamente en el llamado de la función, ejemplo:
function función ( parámetros ) {
return ` $ { parámetros } `
}
console.log ( función ( datos ) )
Si hubiera más código después del return, no se ejecutaría porque el código de la función termina al completar la línea de return. function miFuncion(suma1, suma2){
var resultado = suma1 + suma2;
return (resultado);
}
miFuncion(247, 585);
 //regresará e imprimirá en consola: 832

Funciones de primera clase.

Son funciones que se pueden almacenar en una variable como un valor o como la variable en si.
*Se utiliza const como almacenamiento para las funciones ya que las funciones no son modificables.
const variable = función() guarda la ejecución de la función, es decir el dato resultante de la función.
const variable = función guarda la función para poder utilizar la variable como una función. cont sumaDos = miFuncion;
sumaDos(247, 585);
 //imprimiría: 832
const sumaTres = miFuncion(247, 585);
console.log(sumaTres);
 //imprimiría: 832

Funciones anónimas.

Son funciones que no tienen nombre, se les indica los parámetros que se utilizarán y la acción que se ejecutará, ejemplo:
const variable = function(parámetros){
acción
}
//Se manda a llamar
variable(datos) const bienvenida = function(nombre){
console.log(`Bienvenide ${nombre}!`);
}
 //mandar a llamar
binvenida("Cosmo");

Auto-ejecución

Se pueden invocar las funciones por si solas, se tienen que colocar encerradas en paréntesis para indicarle a JavaScript que se le colocará una expresión, ejemplo:
( function ( a, b ) {
console.log ( a + b )
} ( 4, 5 ) )

Funciones de flecha.

Retornan datos reduciendo el código de una función. Cuando tiene un solo parámetro se pueden omitir los paréntesis donde se colocan (es opcional), pero cuando tenga más de uno o ningún parámetro se colocan los parentesis, ejemplo:
Se puede colocar de una manera muy reducida el código de una función como operaciones matemáticas o impresiones de texto:
const variable = ( parámetro, parámetro ) => código simple
Se pueden colocar llaves para agregarle a la función código de comparación o más complejo:
const variable = parámetro => {
código lógico
} const despedida = apodo => `Chau ${apodo}`
console.log(despedida("Calamardo"))
const despedida = () => console.log("Hasta luego!")
despedida();
