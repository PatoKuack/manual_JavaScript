String

String (Slice).

Parte un texto eliminando las partes indicadas, del primer carácter hasta el enésimo carácter indicado en el primer parámetro y del segundo parámetro hasta el último carácter, ejemplo:
let variable1 = "string"
let variable2 = variable1.slice ( parámetros )
Ejemplo con valores:
let pizza = "La pizza de don cangrejo, es la mejor pizza"
let rodajas = kk.slice ( 12, 37 )
En este caso elimina los carácteres del primero al duodécimo y del trigésimo séptimo hasta el final.

String (Split).

Divide el texto en letras si no se coloca algún parámetro, también puede dividir entre palabras tomando como referencia de separación el carácter que se coloca de parámetro, ejemplo:
let variable2 = variable1.split ( "caracter" )
Ejemplo con valores:
let porpalabra = kk.split ( " " )

String (trim).

Remueve los espacios en blanco solamente de los extremos de un string, ejemplo:
let variable = " string ";
variable.trim();
//resultado: " string " -> "string"

String (StartsWith).

Transforma en mayúsculas todo un string, ejemplo:
let variable2 = variable1.startsWith("valor")
Ejemplo con valores:
let url_hash = location.hash.startsWith("#oso")
location entrega un string con la url actual.
hash agrega el hash de la url actual.

Search.

Necesita una expresión regular (que es una serie de carácteres que simboliza algo que el lenguaje entienda y realice una búsqueda).

Método to... (toLowerCase).

Transforma en minúsculas todo un string, ejemplo:
let variable2 = variable1.toLowerCase()
Ejemplo con valores:
let minus = kk.toLowerCase()

Método to... (toUpperCase).

Transforma en mayúsculas todo un string, ejemplo:
let variable2 = variable1.toLowerCase()
Ejemplo con valores:
let mayus = kk.toUpperCase()
