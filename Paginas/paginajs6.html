Colecciones de datos.

Array.

Es una zona de almacenamiento contiguo que contiene una serie de elementos del mismo tipo.
Sus sintaxis es de las siguientes maneras: let array-de-valores = [ valor0, valor1 ] Con la instrucción array[ número-de-valor ] se obtiene el valor que esté ubicado en la posición especificada, siendo que el primer valor está en la posición 0, el segundo valor en la posición 1, etc., ejemplo: console.log(menu[0])
let array-de-objetos = [
{ // objeto 0
atributo1 : valor1,
atributo2 : valor2
},
{ // objeto 1
atributo3 : valor3,
atributo4 : valor4
}
] Con la instrucción array[ número-de-conjunto ].nombre-de-atributo se obtiene el valor que esté contenido en el atributo especificado y en el objeto ubicado en la posición especificada, siendo que el primer objeto está en la posición 0, el segundo objeto en la posición 1, etc., ejemplo: console.log(libreria[0].nombreLibro)
Se puede escribir directamente en la consola el nombre del array para visualizarlo.

Métodos

Array (push).

Agrega nuevos datos a un array.
Se escribe la función, con un punto se indica que se hará un llamado a un método, se coloca el método deseado y después se colocan los parámetros, ejemplo:
let variable = [ valores ]
variable.push( elemento nuevo )
Se puede guardar la sentencia en otra variable y se obtiene el número de datos que contiene el array, ejemplo:
let variable2 = variable1.push( elemento nuevo )
// la variable2 tendrá un valor numérico el cual será el número de valores que contenga el array, es decir su tamaño.
Ejemplo con valores:
let buffete = [ "gelatina de moho", "telarañas cristalizadas", "aderezo de huesos", "sangre bajo en hierro", "sanguijuelas recién alimentadas"]
let especial = buffete.push( "feto en su jugo" )

Array (unshift).

Funciona igual que push, pero envía el string indicado dentro de su parámetro al inicio del array y no al final.

Array (pop).

Elimina el último valor de un array.
Si se guarda la sentencia de eliminación en una variable, ésta variable contendrá el dato que se está eliminando, ejemplo:
let variable2 = variable1.pop()
Ejemplo con valores:
let agotado = buffete.pop( )

Array (shift).

Funciona igual que pop, pero elimina el string indicado dentro de su parámetro solamente si ése string está al inicio del array.

Array (slice).

El método slice duplica el array, en el parámetro se indica desde que posición, hasta que posición se debe partir.
Este método NO modifica el arreglo original como los métodos anteriores y aparte nos devuelve un arreglo extra.
let variable2 = variable1.slice( 0 , 2 )
//en esté caso nos copiará los datos de la posición 0 y 1 en un array guardado en la variable2
Ejemplo con valores:
let postres = buffete.slice( 0, 2 )

Array (splice).

Agrega, elimina o remplaza elementos del array modificando su contenido. let variable = ['valor1', 'valor3', 'valor4', 'valor5'];
variable.splice(1, 0, 'valor2', 'valorN');
//se posiciona después del primer valor
//elimina cero elementos después de ese posicionamiento
//inserta los valore especificados (valor2, valorN) después de su posición.
variable.splice(-2, 1);
//se posiciona después del segundo elemento (no difiere entre positivos y negativos)
//elimina el elemento que le precede (solo elimina 1 valor).
variable.splice(2);
//se posiciona después del segundo elemento y elimina todos los elementos que le preceden.

Array (forEach).

Significa "para cada elemento". Este método no nos retorna valores ni los guarda. Necesita como parámetro una función con su propio parámetro y su código, el parámetro de la función dentro del forEach hará q se aplique el código de la función a cada uno de los datos obtenidos, ejemplo:
let variable1 = [ datos ]
variable1.forEach ( function ( variable2 ){
código // aplica el código a cada uno de los datos del array "variable1"
} )
o también se puede declarar de la siguiente forma:
let variable1 = [ datos ]
variable1.forEach ( variable2 => código simple )
Ejemplo con valores:
let i = 1
buffete.forEach ( platillos => (console.log( `Platillo ${i}: ${platillos}` ), i++ ) )

Array (map).

Funciona igual que forEach solo que este método crea un array nuevo retornando cada uno de los valores modificados del array original, ejemplo:
let variable2 = variable1.map ( función => valor modificado )
En este caso, variable2 es un array vacío que se va llenando con cada valor retornado que genere cada valor del array.
Ejemplo con valores:
let i = 1
let menu = buffete.map ( platillos => `Platillo ${i++}: ${platillos}` )

Array (filter).

Este método crea una copia del array original sin modificarlo, trabaja con elementos booleanos por lo que se utilizan comparaciones para obtenerlos, filtra los elementos de un array según una comparación, mostrando los elementos que devuelvan un true, ejemplo:
let variable1 = [ datos ]
let variable2 = variable1.filter ( función => comparación )
Ejemplo con valores:
let vamplillos = buffete.filter ( sangre => sangre === "sangre bajo en hierro" || sangre === "sanguijuelas recién alimentadas" )

Array (find).

Encuentra el primer dato del array que cumpla con la comparación, ejemplo:
let variable1 = [ datos ]
let variable2 = variable1.find ( función => comparación )
Ejemplo con valores:
let nombre = buffete.find ( nplatillo => nplatillo.length > 15 )

Array (includes).

Verifica que el valor especificado se encuentre entre los datos del array y nos devuelve un valor booleano, ejemplo:
let variable2 = variable1.includes ( dato )
Ejemplo con valores:
let especifico = buffete.includes ( "feto en su jugo" )

Array (some).

Comprueba si al menos algún elemento del array cumple la comparación y devuelve un booleano, ejemplo:
let variable2 = variable1.some ( función => comparación )
Ejemplo con valores:
let nombre2 = buffete.some ( typeplatillo => typeof typeplatillo === "number" )

Array (every).

Valida si todos los elementos del array cumplen con la condición y devuelve un booleano, ejemplo:
let variable2 = variable1.every ( función => comparación )
Ejemplo con valores:
let nombre3 = buffete.every ( typeplatillo => typeof typeplatillo === "string" )

Array (indexOf).

Encuentra el índice del valor que se le pasó por parámetro y regresa el índice si lo encuentra. Los índices son el número de posición de un elemento en el array y van de 0 a n, sino encuentra dicho valor (índice) entonces regresa '-1', ejemplo:
let variable2 = variable1.indexOf ( comparación )
Ejemplo con valores:
let indice = buffete.indexOf ( "aderezo de huesos" )

Array (reduce).

Funciona como un ciclo generando una operación entre todos los valores de un array, es decir suma todos sus valores, divide, etc.
Recibe una función como argumento, esta función debe contener dos parámetros, uno donde se vaya acumulando el resultado de las operaciones y otro donde reciba los nuevos valores. Ejemplos con valores:
function calcMediaAritmetica(lista) {
//se inicializa una función para mandarla a llamar y ejecutar todo el código del reduct.
| const sumaLista = lista.reduce(
| //a el array que se reciba como argumento al llamar la función se le aplicará el reduct y se guardará en la constante "sumaLista".
| | function (valorAcumulado = 0, nuevoElemento) {
| | //se declara la función que recibe como parámetro el método "reduct" con sus propios dos parámetros, uno de estos dos obtiene un valor por default de 0 si es que no recibe un valor.
| | | return valorAcumulado + nuevoElemento;
| | | //retorna la indicación de que se tiene que hacer una suma con los valores del array.
| | }
| );
| const promedioLista = sumaLista / lista.length;
| return promedioLista;
| //se obtiene el promedio y se retorna.
}
const array1 = [1, 2, 3, 4];
//se declara el array.
const multiplicador = (accumulator, currentValue) => accumulator * currentValue;
//se crea la función para el parámetro del método "reduct" junto con su instrucción de multiplicar.
// 1 * 2 * 3 * 4
console.log(array1.reduce(multiplicador));
//se imprime en consola el array con el método reduct y sus parámetros guardados en la constante "multiplicador".
//el resultado será 24.
