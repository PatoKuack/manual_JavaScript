React portals.

Permiten teletrasportar componentes a un nodo de HTML distinto al nodo donde hace render el resto de la aplicación..

React renderiza toda la aplicación gracias al elemento con el id="root" del archivo src/index.js, con los React portals se pueden renderizar los componentes de la página en otro lugar, en otro nodo y aún así se pueden seguir comunicando los estados de los nodos.

src/Modal/index.js

import React from 'react';
import ReactDOM from 'react-dom';

function Modal() {
 return ReactDOM.createPortal(
  /* Aquí se pueden colocar elementos o componentes para mostrar algo especifico y estatico o algo dinamico que sea reutilisable */
 );
}

Después se colocan los argumentos:

  1. En el primer argumento se coloca lo que se mandará en el portal (en el siguiente ejemplo será un children).
  2. En el segundo argumento se coloca el nodo del código HTML a donde se mandará el contenido del componente modal.
src/Modal/index.js

import React from 'react';
import ReactDOM from 'react-dom';

function Modal({ children }) {
 return ReactDOM.createPortal(
  children,
  document.getElementById('modal')
 );
}
 export { Modal };
