Objetos.

Es una colección de datos relacionados con múltiples variables de diferente tipo y con sus respectivos valores.

Se crea como una variable normal y se colocan sus atributos entre llaves, ejemplo:

let objeto = {
atributo1 : valor1,
atributo2 : valor2
} Se puede escribir directamente en la consola el nombre del objeto para visualizarlo, ejemplo: objeto
Se puede obtener el valor de cada atributo con la sentencia objeto["atributo1"] o objeto.atributo1
Se puede obtener el valor de todos los atributos con la sentencia Object.values(objeto), se puede guardar el resultado en una variable e imprimirlo en consola.
Se puede obtener los atributos del objeto con la sentencia Object.keys(objeto), si el resultado devuelve un método vacío significa que el objeto está vacío.
Se puede obtener el valor de un atributo dentro del objeto con la palabra reservada this, ejemplo:
var videojuego = {
nombre: "Stardew Valley",
desarrollador: "Eric "ConcernedApe" Barone",
publicador: "Chucklefish Games"
lanzamiento: function(){
console.log(`${this.nombre} fue lanzado en 2016`);
} }
//se manda a llamar la función del atributo del objeto así:
videojuego.lanzamiento();

Existen formas más complejas, pero reducidas de obtener datos del objeto:

function myFunction(myObject) {
 for (let myElement in myObject) {
  console.log(`the ${myElement} is: ${myObject[myElement]} =D`);
 }
}
 // imprimirá cada una de las variables con su respectivo valor del objeto especificado.

Con la sentencia Object.entries(objeto) se convierte cada atributo con su respectivo valor en un array y todos los arrays creados por los atributos se guardarán en conjunto dentro de un único array.

Con la sentencia Object.entries(objeto)[0] se obtiene el primer array del array de arrays, es decir, convierte el primer atributo y su valor en un array, ejemplo: Array [ "atributo1", "valor1" ]

Con la sentencia Object.entries(objeto.atributo) se convierte cada caracter del atributo especificado en un array y todos los arrays creados por los caracteres se guardarán en conjunto dentro de un único array.

Con la sentencia Object.assign(objeto1, objeto2) se puede unir los objetos colocados en el parámetro creando un nuevo objeto con todos los atributos y sus valores.

Math.

Nos permite obtener valores matemáticos como:
Math.sqrt(valorNumerico)
Devuelve la raíz cuadrada del valor numérico, pero si es un valor numérico negativo devuelve NaN.
Math.pow(base, exponente)
Devuelve el valor numérico "base" elevado a la potencia especificada en el segundo parámetro.
Math.PI
Esta instrucción contiene el valor de PI, puede guardarse en una variable.
Math.round(valorNumerico)
Redondea números con fracciones (1.50 a 2; 1.49 a 1; -1.50 a -1; -1.51 a -2).
Math.ceil(valorNumerico)
Redondea números con fracciones al mayor (1.05 a 2; -1.9 a -1).
Math.floor(valorNumerico)
Redondea números con fracciones al menor (1.9 a 1; -1.05 a -2).
Math.trunc(valorNumerico)
Redondea números con fracciones truncando su valor fraccional (-1.03 a -1; 5.99 a 5).
Math.min(valNum, valNum, valNum)
Devuelve el valor más pequeño de los valores ingresados como argumento.
Math.max(valNum, valNum, valNum)
Devuelve el valor más grande de los valores ingresados como argumento.
Math.random()
Devuelve un valor aleatorio entre 0 y 1 (jamás dará 1 pero si 0).
Ejemplo para obtener un valor aleatorio entre dos valores. function getRandomArbitrary(min, max){
return Math.random() * (max - min) + min;
}

Date.

Nos permite obtener valores de fechas como .getFullYear(), .getMinutes(), getUTCDate(), etc., ejemplo:
const variable = new Date() //es necesario definir una fecha
console.log(variable.getUTCDate( )) // aquí se manipula la fecha

Objetos: función constructora.

Podemos generar una función con parámetros que sean sus atributos para poder usarlo como plantilla para crear objetos, ejemplo:
function función(atributo1, atributo2){
this.atributo1 = atributo1;
this.atributo2 = atributo2;
}
//uso:
//los valores colocados llevan comillas si son strings, pero no las llevan si son números o booleanos.
var objeto1 = new función("valor1", valor2);
